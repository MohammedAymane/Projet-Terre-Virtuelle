## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/MohammedAymane/navisu4d

# Access
$ cd navisu4d

# Install dependencies
$ npm install

# Run the project
$ npm start

# The server will initialize in the <http://localhost:3000>
```
